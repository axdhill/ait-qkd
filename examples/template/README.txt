This is the AIT QKD module template (for AIT QKD R10 v9.9999.8)
===============================================================

Be sure to have the AIT QKD framework (https://sqt.ait.ac.at/software/projects/qkd) compiled and installed.

How to compile this template
----------------------------

Create a build folder, step into this build folder and invoke the cmake - make sequence:

    $ mkdir build
    $ cd build
    $ cmake ..
    ...
    $ make
    ...


Tweak the template
------------------

Check the sources especially in the bin/ folder. Here substiute each appearance of MY_MODULE, my_module and my-module accordingly to your liking.
